## 🚀 CÓMO USAR

### Instalación Rápida

```bash
# Opción 1: Git
git clone <repo-url>
cd petcare-clinic

# Opción 2: Servidor local
python -m http.server 8000

# Opción 3: Abrir en navegador
http://localhost:8000
```

### Personalización

1. **Colores**: `styles.css` línea 1-60
2. **Nombre**: `index.html` logo y title
3. **Servicios**: Editar `<article class="service-card">`
4. **Equipo**: Editar `<article class="team-member">`
5. **Contacto**: Cambiar dirección, teléfono, email

### Publicación

- GitHub Pages: Gratuito, automático
- Netlify: Drag & drop, CDN incluido
- Hosting tradicional: FTP upload
