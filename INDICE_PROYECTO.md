## 🎓 LECCIONES APRENDIDAS

### Mejor Práctica #1: Variables CSS
```css
/* En lugar de repetir colores */
✅ --color-primary: #10b981;
.btn { background: var(--color-primary); }

❌ .btn { background: #10b981; }
.nav { background: #10b981; }
```

### Mejor Práctica #2: BEM Naming
```css
/* Estructura clara y escalable */
✅ .service-card__title
.service-card__icon
.service-card--featured

❌ .service-title
.serviceCardTitle
.service_card_featured
```

### Mejor Práctica #3: Grid + Flexbox
```css
/* Responsive automático */
✅ Grid para layouts macro (3 cols → 1)
✅ Flexbox para alineación micro

❌ Floats (deprecated)
❌ Absolute positioning (inflexible)
```

### Mejor Práctica #4: Vanilla JS
```javascript
/* Clases modulares sin deps */
✅ class MobileNavigation { }
✅ Zero dependencies

❌ jQuery (deprecated)
❌ Múltiples librerías
```
